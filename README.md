# Todo | FSK

To run the app you need to follow the follow steps:

### Start the API

```bash

    $ cd server
    # install the dependencies
    $ yarn

    # run the migrations
    $ yarn run knex:migrate

    # start the server
    $ yarn start
```

### Start the webapp

```bash

    $ cd web

    # install the dependencies
    $ yarn

    # start the server
    $ yarn start
```

and... thats it :)
